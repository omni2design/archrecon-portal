## 🧠 Overview

A web-based client portal designed for ArchRecon to streamline project visibility, file access, and service requests.

This project translates complex AEC workflows into an intuitive digital interface, improving how clients interact with project data.

---

## ⚙️ Features

* Project dashboard with structured navigation
* File viewer for drawings, plans, and documents
* Request service flow for new projects
* Clean, scalable UI system using Tailwind

---

## 🧱 Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* Vercel (deployment)

---

## 🎯 Goal

Design and prototype a scalable client portal that improves:

* Client communication
* Project transparency
* Workflow efficiency

---

## 🧠 Product Thinking

This project explores the intersection of architecture and product design by translating spatial workflows into user-centered digital tools.
